BaleSupply (PHP MVC + MySQL)

Instalación rápida:
1) Copiar .env.example a .env
2) Ejecutar database/migrations/001_init.sql
3) Ejecutar database/seeds/001_seed.sql
4) Abrir http://localhost/BaleSupply/public/
